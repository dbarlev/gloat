# Wellcome

## How to run
1. Clone the repository
2. Open it using vsCode or any other IDE
3. In terminal/cmd run 'yarn install'
4. Run 'yarn start'
5. ENJOY! :D

## 3rd party libraries
Router - @reach/router.
UI - antd.
Icons - @ant-design/icons.
Style - styled-components.
Ajax calls - axios-hooks.
