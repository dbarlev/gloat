# Wellcome

## How to run
1. Clone the repository
2. Open it using vsCode or any other IDE
3. In terminal/cmd run 'yarn install'
4. Run 'yarn start'
5. ENJOY! :D

## 3rd party libraries
1. Router - @reach/router
2. UI - antd \
3. Icons - @ant-design/icons
4. Style - styled-components
5. ajax - axios-hooks
